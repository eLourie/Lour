# Machine Learning Basics

## Supervised learning

Supervised learning trains a model on labelled examples. Classification predicts
discrete categories; regression predicts continuous values. Common algorithms
include logistic regression, decision trees, and gradient-boosted ensembles.

## Overfitting

A model overfits when it memorises training noise and fails to generalise.
Regularisation, cross-validation, and more data all reduce overfitting.

## Embeddings

An embedding maps a token, sentence, or document into a dense vector so that
semantically similar items lie close together. The bge-m3 model produces 1024
dimensional multilingual embeddings.
